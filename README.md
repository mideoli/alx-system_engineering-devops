# Alx System Engineering Devops project

This repository was created for Project 205 at ALX SE program, the project is intended to make sure we understand the basics of the GNU/Linux shell, in order to be able to explain:

- What is the Shell
- The basics of nagivation on Shell <code>(cd, pwd, ls)</code>
- Loooking around <code>(ls, less, file, ln)</code>
- Manipulating files <code>(cp, mv, rm, mkdir)</code>
- Working with commands <code>(type, which, help, man)</code>
- Reading <code>man</code> pages

As per the general requirements:
* Allowed editors: vi, vim, emacs;
* All scripts were tested on Ubuntu 20.04 LTS;
* All scripts should be exactly two lines long <code>($ wc -l file should print 2)</code>;
* All files should end with a new line;
* The first line of all your files should be exactly <code>#!/bin/bash</code>
* A README.md file at the root of the repo, containing a description of the repository
* A README.md file, at the root of the folder of this project, describing what each script is doing
* It wasn't allowed to use backticks, &&, || or ;
* All scripts must be executable. To make your file executable, use the chmod command: <code>chmod u+x file</code>.
